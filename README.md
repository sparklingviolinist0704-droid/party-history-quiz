# 🚩 党史学习自测题库

中国共产党历史互动学习平台 —— 单/多选题随机自测，错题本辅助记忆。

## ✨ 功能

- **单项选择题** — 122 题，点击即判，即时反馈
- **多项选择题** — 43 题，勾选全部正确答案后提交
- **随机出题** — 每次进入题库自动打乱顺序
- **错题库** — 自动收录错题，支持移除/清空，localStorage 持久存储
- **记忆提示** — 答错时显示正确答案 + 记忆方法引导
- **键盘快捷键** — A/B/C/D 选择选项，Enter 提交/下一题
- **完成统计** — 正确率环形图 + 正确/错误统计

## 🚀 使用方式

双击 `index.html` 即可在浏览器中使用，无需任何服务器。

或在线访问：https://sparklingviolinist0704-droid.github.io/party-history-quiz/
